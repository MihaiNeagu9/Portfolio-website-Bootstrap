# Portfolio Website

Personal portfolio website built with HTML, SCSS, Bootstrap 5, and vanilla JavaScript.

## Tech Stack

- HTML5
- SCSS (Sass)
- Bootstrap 5
- Font Awesome
- Lightbox

## Local Development

Prerequisites:

- Node.js
- npm

Install dependencies:

```bash
npm install
```

Build SCSS once:

```bash
npm run sass:build
```

Watch SCSS during development:

```bash
npm run sass:watch
```

Open `index.html` in your browser (or use a local static server / Live Server).

## Project Structure

- `index.html` - main page
- `scss/` - SCSS source files
- `css/` - compiled CSS
- `js/` - custom and vendor JavaScript
- `images/` - image assets
- `webfonts/` - font files
